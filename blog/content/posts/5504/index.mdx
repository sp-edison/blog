## 결정구조에 대한 기본 정의

<div style='display:flex; position:relative; margin-bottom:20px; justify-content: center;'>

Crystal 구조의 요소는 크게 두 가지로 나뉜다.

하나는 동일하게 반복되는 원자들의 그룹인 basis이며, 다른 하나는 basis가 translational vector($\vec{a}_1,\vec{a}_2,\vec{a}_3$)에 의해 반복되어 놓이는 위치인 lattice이다.

이 두 요소가 합쳐지면 결정구조를 아래의 그림과 같이 결정구조를 이루게 된다.

이때, $\vec{r}'=\vec{r}+u_1\vec{a}_1+u_2\vec{a}_2+u_3\vec{a}_3$(※ $u_1$, $u_2$, $u_3$ 는 정수)

![image](./images/0.jpg)

각각의 평행이동으로 결정을 표현하는 것이 가능한 단위격자를 Unit Cell이라고 하며, Lattice를 형성할 수 있는 가장 작은 unit cell을 Primitive Cell라고 정의한다.


![image](./images/1.jpg)

## 결정구조에 대한 결정면 결정 절차

### 결정면 결정 절차 구하기 예시

각 lattice vector와의 절편이 다음과 같이 있다.
$$
(2a_1 , 3a_2 , 2a_3)
$$



각 계수인 2, 3, 2에 대한 역수를 취한다.
$$
(\frac{{1}}{2} \,, \frac{{1}}{3} \,, \frac{{1}}{2})
$$



각 계수의 최소공배수인 6을 역수에 곱하여 값을 도출한다.
$$
(3\; 2\; 3)
$$


$$
∴ index \, of\,  plane = (3\; 2\; 3)
$$



#### Case : (1 0 0)

각 lattice vector와의 절편 : $(1,∞,∞)$

각 계수의 역수를 취함 : (1 0 0)

최소공배수를 곱함 : (1 0 0)
$$
∴ index \, of\,  plane = (1\, 0\, 0)
$$



![image](./images/9.jpg)

#### Case : (1 1 0)

각 lattice vector와의 절편 : $(1,1,∞)$

각 계수의 역수를 취함 : (1 1 0)

최소공배수를 곱함 : (1 1 0)
  
$$
∴ index \, of\,  plane = (1\, 1\, 0)
$$



![image](./images/10.jpg)

#### Case : (1 1 1)

각 lattice vector와의 절편 : (1, 1, 1)

각 계수의 역수를 취함 : (1 1 1)

최소공배수를 곱함 : (1 1 1)
$$
∴ index \, of\,  plane = (1\, 1\, 1)
$$



![image](./images/11.jpg)

#### Case : (2 0 0)

각 lattice vector와의 절편 : $(\frac{1}{2},∞,∞)$$

각 계수의 역수를 취함 : (2 0 0)

최소공배수를 곱함 : (2 0 0)

$$
∴ index \, of\,  plane = (2\, 0\, 0)
$$



![image](./images/12.jpg)

#### Case : $(\overline{1} 0 0)$

각 lattice vector와의 절편 : $(-1,∞,∞)$$

각 계수의 역수를 취함 : (-1 0 0)

최소공배수를 곱함 : (-1 0 0)
$$
∴ index \, of\,  plane = (\overline{1}\, 0\, 0)
$$



![image](./images/13.jpg)
